# Products API

A simple RESTful API built with Node.js and Express for managing a product catalog. Data is persisted in a local JSON file (`products.json`).

## Features

- **List Products**: Retrieve all products.
- **Filter**: Filter products by category or subcategory.
- **Search**: Search for products using keywords.
- **Create Product**: Add new products with auto-generated IDs.
- **Persistence**: Data is saved to `products.json`.

## Prerequisites

- Node.js installed on your machine.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

Start the server:

```bash
node server.js
```

The server will start on `http://localhost:9000`.

## API Endpoints

### 1. Get Products

**URL:** `/products`  
**Method:** `GET`

**Query Parameters:**
- `category` (optional): Filter by product category (e.g., `Apparel`).
- `subcategory` (optional): Filter by subcategory (e.g., `Hat`).
- `search` (optional): General search term that matches against the product object.

**Example:**
`GET http://localhost:9000/products?category=Electronics`

### 2. Create Product

**URL:** `/products`  
**Method:** `POST`  
**Body:**
Provide the product details. The `id` is automatically generated by the server.

```json
{
  "name": "New Product Name",
  "category": "Electronics",
  "subcategory": "Laptop",
  "price": 999.99,
  "currency": "USD",
  "stock": 10,
  "rating": 4.5
}
```
